# 📖 About the Project

**Electraze 2K25** is a modern event registration website created for the **Electronics and Communication Engineering Department Fest**.

Participants can register online, select technical and non-technical events, and securely submit their information using **Supabase** as the backend database.

---

# ✨ Features

✅ Beautiful Landing Page

✅ Responsive Registration Form

✅ Real-time Data Storage using Supabase

✅ Bootstrap UI Components

✅ Event Selection with Checkboxes

✅ Form Validation

✅ Modern Animated Design

✅ Mobile Friendly

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/vinishav2405-cyber/Electraze-2K25.git
```

Move into the project folder

```bash
cd Electraze-2K25
```

Open `index.html` in your browser.

---

# 🔥 Supabase Configuration

Create a `.env` file (or configure your project appropriately) and provide your own Supabase credentials.

```env
SUPABASE_URL=your_project_url
SUPABASE_ANON_KEY=your_public_anon_key
```

> ⚠️ **Never upload your Supabase Service Role Key or any secret keys to GitHub.**
